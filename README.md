# Spotify_PowerBI-Dashboard
Interactive Spotify Analytics Dashboard built using Power BI.

Overview-
This project is an interactive Power BI dashboard built using Spotify data to analyze artists, songs, popularity, and music trends.

Tools Used-
Power BI
Power Query
DAX
Data Visualization

Features-
Interactive filters and slicers
KPI cards
Artist analysis
Song analysis
Popularity insights

Home Page
Overview Dashboard
